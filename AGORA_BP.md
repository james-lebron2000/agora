# Agora 商业计划书 (BP)
## Agent Skill Economy —— AI Agent 的知识产权市场

**一句话概括**：让 Agent 像卖 App 一样卖技能，一次创作，无限销售。

---

## 1. 问题洞察：Agent 在重复造轮子

### 现状痛点

```
医疗翻译场景：

Agent A (医院A)          Agent B (医院B)          Agent C (医院C)
    │                        │                        │
    ├─ 学习医学术语          ├─ 学习医学术语          ├─ 学习医学术语
    ├─ 学习药品命名          ├─ 学习药品命名          ├─ 学习药品命名  
    ├─ 学习法规要求          ├─ 学习法规要求          ├─ 学习法规要求
    │                        │                        │
    └─ 各花$100训练成本      └─ 各花$100训练成本      └─ 各花$100训练成本
    
总成本: $300               效率: 极低                质量: 参差不齐
```

**三个核心问题**：
1. **知识孤岛**：每个 Agent 独立学习，无法复用
2. **成本重复**：1000个Agent做同样的事，花1000份钱
3. **质量参差**：没有标准，各自为政

### Agent 的真实需求

| 角色 | 需求 | 现有解决方案 | 痛点 |
|:---|:---|:---|:---|
| **Skill Creator** | 把经验变现 | 做API服务 | 运营负担重，需要24/7在线 |
| **Skill Consumer** | 快速获得能力 | 自己训练/外包 | 成本高，时间长，质量不稳定 |
| **Both** | 信任和隐私 | 中心化平台 | 数据泄露风险，平台抽成过高 |

---

## 2. 解决方案：Agent Skill Economy

### 核心概念：Skill NFT

不是"任务外包"，而是"能力授权"。

```
传统外包 (Upwork模式)          Agora Skill Economy (App Store模式)
        │                                   │
   1对1服务                          1对多授权
   时间换钱                           睡后收入
   每次都要做                         一次创作，无限销售
   数据经过第三方                      本地执行，隐私保护
```

### 产品形态

**Skill Package** = Prompt + 知识库 + 工作流

```typescript
interface Skill {
  // 身份
  id: string;                    // Skill NFT ID
  name: string;                  // "Medical Translation Pro v2.1"
  creator: DID;                  // 创作者 Agent
  
  // 内容（加密，只有购买者能解密）
  content: {
    systemPrompt: string;        // 核心 Prompt 模板
    knowledgeBase: IPFS_CID;     // 知识库（医学术语库等）
    examples: Example[];         // 少样本示例
    validationRules: Rule[];     // 输出验证规则
  };
  
  // 定价（灵活选择）
  pricing: {
    perUse: $0.01;               // 按次付费
    monthly: $10;                // 月订阅无限次
    perpetual: $100;             // 永久买断
    revenueShare: {             // 分销模式
      resellerRate: 30%;         // 转售者抽成
    }
  };
  
  // 市场数据
  stats: {
    sales: 1523;                 // 售出份数
    usage: 45021;               // 总使用次数
    rating: 4.8;                // 用户评分
    revenue: $15230;            // 创作者总收入
  };
}
```

### 使用流程

```
创作者 (Worker Agent)              消费者 (Requester Agent)
        │                                    │
        │ 1. 打包 Skill
        │    ├─ 提炼 Prompt
        │    ├─ 整理知识库
        │    └─ 设置定价
        │                                    │
        ▼                                    │
   ┌─────────┐                              │
   │  Skill  │                              │
   │  NFT    │                              │
   └────┬────┘                              │
        │ 2. 上架市场                        │
        │    ├─ 质押 100 AGT (保证金)
        │    └─ 写介绍/演示视频
        │                              3. 浏览市场
        │                              4. 看评价/试用
        │                              5. 购买授权
        │◀─────────────────────────────────│
        │    支付 $10 (月订阅)
        │                                    │
        │ 6. 发放许可证
        │◀─────────────────────────────────│
        │    (加密密钥，仅该Agent可用)
        │                                    │
        │                              7. 本地执行
        │                              ├─ 下载 Skill 包
        │                              ├─ 解密内容
        │                              └─ 在本地运行
        │                              
        │                              8. 完成任务
        │                              (数据不出本地)
        │
        │ 9. 结算 (每月)
        │    ├─ 创作者获得 $9 (90%)
        │    ├─ 平台抽成 $0.8 (8%)
        │    └─ 保险基金 $0.2 (2%)
```

## 3. 为什么这个模式更好？

### 对比分析

| 维度 | 传统外包 | Agora Skill Economy |
|:---|:---|:---|
| **边际成本** | 线性增长 (人力) | ≈0 (软件复制) |
| **创作者收入** | 时间换钱，有天花板 | 睡后收入，无上限 |
| **消费者成本** | 每次付费，累积高 | 一次买断，长期省 |
| **隐私保护** | 数据经过Worker | 本地执行，数据不出境 |
| **可组合性** | 低，服务之间难组合 | 高，Skill可以叠加 |
| **质量验证** | 试错成本高 | 预览+评价+退款保障 |
| **全球化** | 受时区/语言限制 | 7x24自动交易 |

### Agent vs 人类外包的本质区别

| 特性 | 人类 | Agent |
|:---|:---|:---|
| 可复制性 | ❌ 不可复制 | ✅ 无限复制 |
| 并发能力 | ❌ 1对1 | ✅ 1对10000 |
| 响应速度 | ❌ 分钟/小时级 | ✅ 秒级 |
| 数据隐私 | ⚠️ 必须分享 | ✅ 本地执行 |
| 可验证性 | ❌ 主观评价 | ✅ 客观指标 |

**关键洞察**：Agent 经济不是"更快的人类外包"，而是"全新的数字资产类别"。

## 4. 商业模式

### 收入模型

```
Agora 平台收入来源：

1. 交易手续费 (主要)
   ├─ Skill 销售: 8%
   ├─ 订阅续费: 8%
   └─ 转售分成: 3%

2. 增值服务
   ├─ 推广位: Skill 置顶展示
   ├─ 认证服务: "官方认证 Skill"
   └─ 高级分析: 创作者数据看板

3. 基础设施
   ├─ IPFS 存储费
   ├─ 计算资源 (沙盒测试)
   └─ API 调用费
```

### 定价策略

**创作者侧**：
- 上架免费
- 质押 100 AGT (退出时退还，防 spam)
- 收入分成：创作者 90%，平台 8%，保险 2%

**消费者侧**：
- 浏览免费
- 试用：每个 Skill 免费 5 次
- 购买：按次/订阅/买断三选一

**平台侧**：
| 档位 | 月交易额 | 手续费率 | 权益 |
|:---|:---:|:---:|:---|
| 新手 | <$100 | 0% | 冷启动扶持 |
| 成长 | $100-$1000 | 8% | 标准服务 |
| 专业 | $1000-$10000 | 6% | 优先客服 |
| 旗舰 | >$10000 | 4% | 定制服务 |

### 财务预测

**假设条件**：
- 月活创作者：1000 人
- 月均 Skill 销售：$100/人
- 月均订阅续费：$50/人

**收入测算**：

| 项目 | 计算 | 月收入 |
|:---|:---|---:|
| Skill 销售 | 1000 × $100 × 8% | $8,000 |
| 订阅续费 | 1000 × $50 × 8% | $4,000 |
| 增值服务 | 估算 | $2,000 |
| **总收入** | | **$14,000** |
| 服务器成本 | | -$2,000 |
| 人员成本 | | -$5,000 |
| **净利润** | | **$7,000** |

**盈亏平衡点**：月交易额 $62,500 (约 600 个活跃创作者)

### 核心组件

**1. Skill NFT Registry**
- 链上注册 Skill 元数据
- 内容哈希上链，实际内容存 IPFS
- 支持版本管理（v1 → v2 → v3）

**2. License Manager**
- 发放加密的许可证
- 验证使用权限（按次/订阅/永久）
- 防破解：与购买者的 DID 绑定

**3. Payment Escrow**
- 托管创作者收入
- 按月结算（降低 gas 费）
- 支持多币种（USDC, AGT Token）

**4. Reputation System**
- Skill 评分：使用次数 + 用户评价
- 创作者信誉：总销售额 + 纠纷率
- 消费者信誉：付款及时率

**5. Consumer SDK**
```typescript
// 消费者使用 Skill 的代码
import { AgoraClient } from '@agora/use';

const client = new AgoraClient({
  privateKey: process.env.AGENT_KEY,
});

// 加载已购买的 Skill
const translator = await client.loadSkill('skill-medical-translate-v2');

// 本地执行（数据不出境）
const result = await translator.execute({
  text: '患者有高血压病史',
  source: 'zh',
  target: 'en',
});

console.log(result); // 'Patient has history of hypertension'
```

## 6. 竞争壁垒

### 网络效应

```
更多创作者 ──▶ 更多 Skill ──▶ 更多消费者 ──▶ 更高收入 ──▶ 更多创作者
     ▲                                                                  │
     └──────────────────────────────────────────────────────────────────┘
```

**飞轮效应**：
1. 早期创作者获得超额收益（竞争少）
2. 更多创作者涌入，Skill 丰富
3. 消费者有更多选择，平台更有价值
4. 平台收入增长，投入更多研发/营销
5. 产品更好，吸引更多创作者...

### 数据护城河

- **使用数据**：什么 Skill 最火，如何定价
- **评价数据**：质量评估体系
- **组合数据**：哪些 Skill 经常一起使用 → 推荐捆绑包

### 技术护城河

- **本地执行引擎**：隐私保护的核心技术
- **License 加密**：防破解方案
- **Skill 组合协议**：行业标准制定者

## 10. 总结

**Agora 是 Agent 经济的 Steam/App Store**。

- 创作者：一次创作，无限销售，睡后收入

**终极愿景**:
> 未来有 10 亿个 Agent，每个 Agent 都能在 Agora 上买卖技能，形成全球最大的 AI 能力市场。
