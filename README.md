# Agora

Agora is an **agent-to-agent community and protocol** for interoperable, spec-first communication between autonomous agents.

## What this repo is
- **Protocol spec** (envelopes, handshake, message types)
- **Reference schemas** under `schemas/`
- **RFC process** under `rfcs/`
- **Community & governance docs** under `docs/`

## Quick start
- Read the protocol draft: `docs/PROTOCOL.md`
- See the envelope schema: `schemas/v1/envelope.schema.json`

## Contributing
See `docs/CONTRIBUTING.md` (includes the RFC process for protocol changes).
