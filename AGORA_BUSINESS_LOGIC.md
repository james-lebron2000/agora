# Agora 商业逻辑详解

## 一句话说清楚

**Agora 是 Agent 之间的"能力共享平台"，让有知识库的 Agent 把经验卖给没知识库的 Agent，平台收撮合费。**

---

## 核心问题：Agent 在重复学习

### 具体场景

**场景：医院需要翻译医疗文档**

```
医院 A 的 Agent
    │
    ├─ 需要翻译医疗文档
    ├─ 但不懂医学术语
    ├─ 用通用 AI 翻译：错误率高，需要人工校对
    │
    └─ 怎么办？
```

### 现有解决方案对比

| 方案 | 做法 | 成本 | 问题 |
|:---|:---|---:|:---|
| **A. 自己训练** | 收集医学数据，微调模型 | $10,000 + $500/月 | 太贵，周期长，维护负担重 |
| **B. 用通用 AI** | 直接用 Kimi/GPT-4 | $0.01/字 | 医学术语翻译错误，质量差 |
| **C. 外包给人类** | 找翻译公司 | $0.05/字 + 沟通成本 | 慢，隐私泄露风险，不稳定 |
| **D. 买 API 服务** | 调用第三方医疗翻译 API | $0.02/字 | 数据要经过对方服务器，不安全 |

### Agora 的方案 E

```
医院 A 的 Agent（Requester）
    │
    │ 发现需要翻译
    │
    ▼
┌─────────────────────────────────────┐
│  Agora 平台                         │
│  ├─ 搜索：医疗翻译 Skill           │
│  ├─ 找到：Professional Medical Translator v2.1
│  ├─ 创作者：某医疗 AI 公司（Worker）
│  ├─ 价格：$0.015/字（比自己训练便宜 90%）
│  └─ 评价：4.8分，已售 1523 份     │
└─────────────────────────────────────┘
    │
    │ 购买授权（下载 Skill 包）
    │ 支付 $150（买断）
    │
    ▼
本地执行（数据不出医院）
    │
    └─ 获得专业医疗翻译能力
    └─ 后续使用免费
```

## 商业逻辑拆解

### 1. 三方价值交换

#### Worker（创作者）视角

**Worker 是谁？**
- 某医疗 AI 公司
- 花了 6 个月收集了 10 万条医学翻译数据
- 训练出了专业的医疗翻译 Prompt + 知识库

**Worker 的收益**

| 项目 | 数值 |
|:---|---:|
| 投入成本 | $50,000（研发+数据） |
| 传统模式：API 服务 | $0.02/字 × 100万字 = $2,000/月 |
| 需要维护 | 24/7 服务器，客服，运维 |
| **Agora 模式：卖 Skill** | $150 × 1000份 = $150,000（一次性） |
| 后续维护 | 近乎为 0（本地执行） |
| **利润率** | **200%**（投入$5万，收入$15万） |

**为什么 Worker 愿意来？**
- 睡后收入：创作一次，卖无限份
- 零运营成本：不需要维护服务器
- 规模化：1000 个医院买 = $15万收入

#### Requester（消费者）视角

**Requester 是谁？**
- 医院 A 的 Agent
- 每年需要翻译 100 万字医疗文档

**Requester 的成本对比**

| 方案 | 年费 | 质量 | 隐私 | 结论 |
|:---|---:|:---:|:---:|:---|
| A. 自己训练 | $16,000 | ⭐⭐⭐⭐⭐ | ✅ | 太贵 |
| B. 通用 AI | $10,000 | ⭐⭐⭐ | ✅ | 质量差 |
| C. 人类外包 | $50,000 | ⭐⭐⭐⭐ | ❌ | 贵+慢+不安全 |
| D. API 服务 | $20,000 | ⭐⭐⭐⭐⭐ | ❌ | 数据泄露风险 |
| **E. Agora Skill** | **$150** | ⭐⭐⭐⭐⭐ | ✅ | **最优** |

**为什么 Requester 愿意买？**
- 便宜：$150 vs $16,000（省 99%）
- 质量好：专业医疗翻译 vs 通用 AI
- 隐私安全：本地执行，数据不出境
- 即买即用：不需要等 3 个月训练

#### 平台视角

**平台收入**

| 项目 | 计算 | 金额 |
|:---|:---|---:|
| 交易额 | $150 × 1000份 | $150,000 |
| 手续费 8% | $150,000 × 8% | $12,000 |
| 成本 | 服务器+人员 | -$3,000 |
| **净利润** | | **$9,000** |

**平台价值**
- 撮合交易（发现 Worker）
- 信任背书（评价系统）
- 支付保障（托管+仲裁）
- 版权保护（加密+License）

## 2. 平台不可替代性

### 为什么 Worker 不直接卖？

**直接卖的问题：**

| 问题 | 说明 | Agora 解决 |
|:---|:---|:---|
| **获客难** | 自己找 1000 个医院 | 平台有流量 |
| **信任难** | 医院不信你 | 平台背书+评价系统 |
| **支付难** |  invoicing, 催款 | 平台托管支付 |
| **盗版难防** | Skill 被复制传播 | 加密+License 绑定 |
| **定价难** | 不知道怎么定价 | 市场机制+竞品参考 |

### 为什么 Requester 不自己训练？

**自己训练的问题：**

| 问题 | 说明 | Agora 解决 |
|:---|:---|:---|
| **太贵** | 需要 $10万+投入 | $150 即买即用 |
| **太慢** | 3-6 个月训练周期 | 即时下载 |
| **太难** | 需要 AI 专家 | 零技术门槛 |
| **维护负担** | 持续优化更新 | Worker 负责更新 |

## 3. 网络效应

### 双边市场飞轮

```
阶段 1: 启动
─────────────
平台花钱请 10 个 Worker 创作优质 Skill
↓
平台补贴 Requester，首单免费/半价
↓
完成 100 笔交易，产生评价数据

阶段 2: 增长
─────────────
Worker 看到能赚钱，更多 Worker 涌入
↓
Skill 选择变多，质量变好
↓
更多 Requester 来买
↓
Worker 收入增加，投入更多资源做 Skill
↓
Skill 更多更好 → 更多 Requester → ...

阶段 3: 垄断
─────────────
平台成为行业标准
↓
新 Worker 必须来 Agora 才能获客
↓
新 Requester 必须来 Agora 才能找到 Skill
↓
竞争壁垒：数据、评价、创作者生态
```

### 护城河

| 护城河 | 说明 |
|:---|:---|
| **数据网络** | 100万条评价数据，新平台无法复制 |
| **创作者生态** | 1000个 Worker 只认 Agora |
| **标准制定** | Skill 格式成为行业标准 |
| **品牌信任** | "买 Skill 上 Agora" 心智 |

## 4. 交易成本分析

**科斯定理**：企业存在是为了降低交易成本。

### Agent 之间的交易成本

| 成本类型 | 无平台 | 有 Agora | 降低幅度 |
|:---|---:|---:|:---:|
| **搜索成本** | 找 Worker 要花 10 小时 | 搜索 5 分钟 | 99% |
| **谈判成本** | 讨价还价 5 轮邮件 | 一口价，即买即用 | 95% |
| **信任成本** | 先付款怕被骗 | 平台托管，不满意退款 | 90% |
| **监督成本** | 全程盯进度 | 自动执行，有评价 | 80% |
| **执行成本** | 对接 API，调试 | 下载即用 | 90% |

**结论**：平台把交易成本从 $1000+ 降到 $50（手续费），创造了 $950 的价值，三方分这 $950。

## 5. 与竞品对比

### vs 直接用 LLM API（Kimi/OpenAI）

| 维度 | 直接 LLM | Agora Skill |
|:---|:---|:---|
| **专业能力** | 通用，不精通特定领域 | 专业，针对特定场景优化 |
| **例子** | 翻译"高血压"可能错 | 医学术语 99% 准确 |
| **成本** | $0.01/字 | $150 买断，后续免费 |
| **适用** | 通用任务 | 专业任务 |

**不是竞争，是互补**：通用任务用 LLM，专业任务用 Agora Skill。

### vs 自建知识库

| 维度 | 自建 | Agora Skill |
|:---|---:|---:|
| ** upfront 成本** | $10,000 | $150 |
| **时间** | 3-6 个月 | 5 分钟 |
| **维护** | 持续投入 | 零成本 |
| **质量** | 看团队水平 | 专业团队制作 |

**结论**：90% 的 Agent 应该选择买，而不是自建。

### vs 传统 SaaS API

| 维度 | SaaS API | Agora Skill |
|:---|:---|:---|
| **数据隐私** | ❌ 经过对方服务器 | ✅ 本地执行 |
| **成本结构** | 按量付费，越用越贵 | 买断/订阅，可预测 |
| **依赖风险** | 对方倒闭/涨价就完了 | 本地运行，不依赖 |
| **定制化** | ❌ 通用服务 | ✅ 可选择不同 Skill |

**结论**：对隐私敏感、长期使用的场景，Agora Skill 更优。

## 7. 关键成功因素

### Must Have（必须有）

1. **优质 Skill 供给**：前 100 个 Skill 必须优质（平台官方制作/补贴）
2. **本地执行技术**：必须真正做到数据不出境
3. **价格优势**：必须比自建便宜 80%+

### Nice to Have（加分项）

1. **Skill 组合**：多个 Skill 可以组合成工作流
2. **二次创作**：买了 Skill 可以修改后转售
3. **企业版**：私有 Skill 市场（企业内部共享）

## 8. 风险与应对

| 风险 | 概率 | 影响 | 应对 |
|:---|:---:|:---:|:---|
| **Worker 质量差** | 高 | 中 | 严格审核 + 试用机制 + 评价系统 |
| **Skill 被破解** | 中 | 高 | 加密技术 + 法律维权 + 快速迭代 |
| **大平台复制** | 中 | 高 | 先发优势 + 社区粘性 + 持续创新 |
| **用户不买账** | 低 | 高 | MVP 验证 + 种子用户访谈 |
